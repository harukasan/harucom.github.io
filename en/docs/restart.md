There are two ways to restart Harucom: pressing <kbd><kbd>Ctrl</kbd>-<kbd>Alt</kbd>-<kbd>Delete</kbd></kbd> on the keyboard, or pressing the RESET button on the board.

Use this when a program becomes unresponsive or when you want to reset the system state.

> Restarting clears all data in memory. Save any data you are editing to a file before restarting.
{: .tip}

## Restart with Ctrl-Alt-Delete

Press <kbd>Ctrl</kbd>, <kbd>Alt</kbd>, and <kbd>Delete</kbd> on the USB keyboard at the same time, and Harucom will restart immediately.

| Key | Action |
|-----|--------|
| <kbd><kbd>Ctrl</kbd>-<kbd>Alt</kbd>-<kbd>Delete</kbd></kbd> | Restart Harucom |

This is the most convenient method for everyday use.

## Restart with the RESET Button

The Harucom Board has a RESET button. Pressing it will restart Harucom.

If Harucom stops responding to keyboard input and you cannot restart with <kbd><kbd>Ctrl</kbd>-<kbd>Alt</kbd>-<kbd>Delete</kbd></kbd>, use the RESET button instead.

> The RESET button is not a power button. To turn Harucom off, unplug the USB-C cable, or turn off the power switch if your board has one.
{: .tip}
